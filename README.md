--- 🎮 UNIVERSAL BEDROCK SHADER INJECTOR (LINUX) ---

Yo bro! Welcome to v1.1. We made it even easier because we know 
you're too lazy to extract files manually. (⌐■_■)

1. PREPARING THE GOODS (EASY AS PIE!)
   - Step A: Get your favorite Shader ".mcpack" file.
   - Step B: Just THROW (copy/paste) that ".mcpack" file directly into 
             the "MaterialsFiles" folder. No need to open it anymore!
   - Step C: Install that same ".mcpack" into your Minecraft as usual.

2. IN-GAME ACTIVATION
   - Open Minecraft -> Settings -> Global Resources.
   - Activate the Shader Pack and MOVE IT TO THE TOP of the list.
   - Seriously, if it's not on top, don't complain to me. =)))

3. RUNNING THE SCRIPT
   - Ensure you have 'unzip' installed (sudo apt install unzip).
   - Make the script executable: chmod +x your_script_name.sh
   - Fire it up: ./your_script_name.sh
   - On the first run, paste your game's 'materials' folder path.
     Example: /home/user/.var/app/com.trench.trinity.launcher/data/.../materials/
   - The script will now AUTOMATICALLY "rip" the .bin files out of 
     your .mcpack and inject them. Magic, right? (づ｡◕‿‿◕｡)づ

4. YOUR OPTIONS
   [1] Inject: The script auto-extracts the .mcpack and pushes shaders to the game.
   [2] Restore: Returns to Vanilla (fixes the game if it looks like a glitchy mess).
   [3] Reset: Deletes the saved path for updates or re-setup.

5. BRO TIPS
   - Only put ONE .mcpack in the folder at a time to avoid confusing the script.
   - Black screen? Use Option [2] and check if your shader version is compatible.
   - Still struggling? Maybe gaming isn't for you, try touching some grass. =)))

Happy Modding! v1.1 is here to save your time.
