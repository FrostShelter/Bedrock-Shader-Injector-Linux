--- 🎮 UNIVERSAL BEDROCK SHADER INJECTOR (LINUX) ---

Yo bro! Read this carefully or your shaders will look like absolute garbage.
Follow these steps to get that sexy Java-like look on Linux Bedrock! (⌐■_■)

1. PREPARING THE GOODS (CRITICAL STEP!)
   - Step A: Open your Shader's ".mcpack" file (use Ark, Engrampa, or any zip tool).
   - Step B: Navigate to: renderer/materials/ inside that pack.
   - Step C: Extract ALL ".bin" files and paste them into the "MaterialsFiles"
             folder (the one sitting right next to this script).
   - Step D: Install that ".mcpack" into your Minecraft as usual.

2. IN-GAME ACTIVATION
   - Open Minecraft -> Settings -> Global Resources.
   - Activate the Shader Pack you just installed.
   - MOVE IT TO THE TOP of the list. Don't forget this, bro! The game needs
     to know who's the boss.

3. RUNNING THE SCRIPT
   - Make it executable: chmod +x your_script_name.sh
   - Fire it up: ./your_script_name.sh
   - On the first run, paste your game's 'materials' folder path.
     Example (Trinity Launcher):
     /home/user/.var/app/com.trench.trinity.launcher/data/mcpelauncher/versions/MC 1.26.3.1/assets/assets/renderer/materials/
   - The script will AUTOMATICALLY backup your vanilla files. You're welcome!

4. YOUR OPTIONS
   [1] Inject: Pushes the shader files into the game.
   [2] Restore: Returns to Vanilla (use this if your game looks like a mess).
   [3] Reset: Deletes the saved path so you can re-setup or update game version.

5. BRO TIPS
   - Update your game? Use Option [3] and re-run the setup.
   - Black screen or invisible blocks? Use Option [2] immediately.
   - Still can't figure it out? Maybe you should just go touch some grass. =)))
